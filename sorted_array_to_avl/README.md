# 0x0D. Array to AVL

This project implements a function that converts a sorted array into a height-balanced AVL Tree using C.

## Requirements

- Ubuntu 14.04 LTS
- gcc 4.8.4 with `-Wall -Werror -Wextra -pedantic`
- Betty style
- No more than 3 functions per file
- No rotations allowed
- No global variables
