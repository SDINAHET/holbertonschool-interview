# UTF-8 Validation

This project implements a method to verify if a sequence of bytes is a valid UTF-8 encoding.

## Usage

```bash
$ ./0-main.py
True
True
False
```

Files
0-validate_utf8.py: The validator function

0-main.py: Test file
